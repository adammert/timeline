# Tech Stack - Timeline Visualizer

## Core Technologies
- **Language:** TypeScript - Providing type safety and modern JavaScript features.
- **Build System:** Vite - Fast build tool and development server with Hot Module Replacement.
- **Markdown Engine:** Marked.js - Robust and efficient Markdown-to-HTML conversion.

## Libraries & Frameworks
- **Exporting (Images):** html2canvas - For capturing the visual timeline as PNG.
- **Exporting (Documents):** jsPDF - For generating PDF versions of the timeline.

## Infrastructure & Environment
- **Runtime:** Browser-based (Client-side only)
- **State Management:** LocalStorage - For persistence without a backend database.
- **Architecture:** Modular ES6 Modules - Each component is isolated and communicates through defined interfaces.
